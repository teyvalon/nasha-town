# TEYVALON AI v2：概率推理与决策模型

基于贝叶斯信念更新的高级 AI 算法，改编自原版阿瓦隆 AI 模型，适配 TEYVALON 规则。

## 1. 概述

AI 基于贝叶斯信念更新，结合公开信息（投票、任务结果）动态调整对其他玩家身份的怀疑程度，并利用期望效用最大化做出组队、投票和任务执行决策。

---

## 2. 信念表示

设总玩家数为 N，深渊玩家数为 E，当前 AI 玩家为 p。

对于其他玩家 j ≠ p，AI 维护一个 **深渊概率**：

```
b_j = P(j 是深渊 | 到目前为止观察到的一切)
```

实际计算中，为了避免概率卡在 0 或 1，使用 **对数几率 (log-odds)**：

```
ℓ_j = ln( b_j / (1 − b_j) )
```

所有更新都在 ℓ_j 上加减一个值，然后再换算回概率：

```
b_j = 1 / (1 + e^(-ℓ_j))
```

## 3. 先验概率初始化

根据 AI 自身的角色，设置每个其他玩家的初始 b_j。所有概率最后会被截断在 [0.001, 0.999]。

| 角色 | 初始化规则 |
|------|------------|
| **哥伦比娅（月神）** | 看到的深渊玩家：b_j = 0.999；镇民：b_j = 0.001 |
| **菈乌玛（咏月使）** | 看到的两人（哥伦比娅和多托雷）：b_j = 0.5；其余玩家 b_j = (E−1)/(N−3) |
| **普通镇民** | b_j = E/(N−1) |
| **深渊玩家（互知）** | 同伴：b_j = 0.999；已知镇民：b_j = 0.001 |

说明：
- 哥伦比娅能看到**所有**深渊玩家，没有例外。
- 菈乌玛看到两个"三月权能"持有者，但不知道谁是真月神、谁是赝月。对这两人初始 0.5，对其余 N−3 人按剩余深渊名额 (E−1) 均分。
- 深渊玩家互相知道身份，因此对同伴取 0.999，对确定的镇民取 0.001。

## 4. 观察更新

### 4.1 任务结果更新

给定一个队伍 S 和观察到的假爆桶数 f。

**假爆桶概率模型**：每个深渊玩家独立以概率 q 放置假爆桶（估计值 q = 0.7）。如果队伍中恰好有 k 个深渊，恰好出现 f 个假爆桶的概率为二项分布：

```
P(f | k) = C(k, f) · q^f · (1-q)^(k-f)    当 f ≤ k
P(f | k) = 0                                当 f > k
```

这比旧的二元 Q(k) 模型更精确：
- 0 假桶 vs 1 假桶 vs 2 假桶各自提供不同强度的信息
- 例如 3 人队出 2 假桶 → 至少有 2 个深渊 → 第 3 人几乎确定是镇民

对于队伍中的每个成员 j（排除自己），我们分别计算"假设 j 是深渊"和"假设 j 是镇民"时出现恰好 f 个假爆桶的概率。

设 S 去掉 j 后的集合为 S'。枚举 S' 中每个子集 T 的身份组合：

```
P(T) = ∏{i∈T} b_i · ∏{i∈S'\T} (1 − b_i)
```

- 如果 j 是深渊，队伍中深渊数为 |T|+1，观察概率 = Σ_T P(T) · P(f | |T|+1)
- 如果 j 是镇民，队伍中深渊数为 |T|，观察概率 = Σ_T P(T) · P(f | |T|)

似然比：

```
LR_j = P(f | j 是深渊) / P(f | j 是镇民)
```

更新对数几率：ℓ_j ← ℓ_j + ln(LR_j)，并重新计算 b_j。

### 4.2 公开投票更新

提案队伍 S 后，所有人公开投票（赞成/反对）。根据投票前的信念，先计算当前队伍的成功概率：

```
P_fail = Σ_{f≥fails_needed} P(恰好 f 个假爆桶)
P_succ = 1 − P_fail
```

定义两个辅助概率：
- 镇民投赞成的概率：p_good = ε + (1−2ε) · σ(α·(P_succ − 0.5))
- 深渊投赞成的概率：p_evil = ε + (1−2ε) · σ(β·(P_fail − 0.5))

其中 σ(x) = 1/(1+e^(-x))，典型参数：α = β = 5，ε = 0.05。

对每一个投票者 i：
- 如果投了赞成，似然比 LR = p_evil / p_good
- 如果投了反对，似然比 LR = (1−p_evil) / (1−p_good)

更新：ℓ_i ← ℓ_i + ln(LR)，然后同步 b_i。

## 5. 哥伦比娅伪装机制

哥伦比娅拥有完美信息（知道所有深渊玩家），但如果行为完美就会被猎月人轻易识别。因此她需要故意"演"，用伪装率（disguise rate）控制犯错频率：

```
disguise_rate(wave) = 0.40 × (1 - wave/4) + 0.12 × (wave/4)
```

- 第 1 波：40%（早期多犯错，反正还不紧急）
- 第 5 波：12%（后期必须认真，不能再送了）

**组队伪装**：以 disguise_rate 的概率，在组队时故意包含一个深渊玩家，使组队模式看起来和普通镇民一样。

**投票伪装**：以 disguise_rate 的概率，翻转自己的真实投票——该赞成时反对，该反对时赞成。

## 6. 行动决策

### 6.1 组队（AI 是队长）

需要选择一个大小符合任务要求的人数 m 的队伍 S。

**镇民队长的目标函数**：

```
U_good(S) = P_succ(S) × P_accept(S)^λ
```

**深渊队长的目标函数**：

```
U_evil(S) = P_fail(S) × P_accept(S)^λ
```

并且深渊必须保证队伍中至少有一个"确定能投假爆桶"的深渊玩家（从 AI 视角 b≈1 的同伴，或者包含自己），否则效用为 0。

P_accept(S) 是提案被多数赞成的概率，可以用每个玩家当前信念下的期望投票行为近似计算。λ 取 0.3。

在全部候选队伍中枚举，选择目标函数最大的队伍。

### 6.2 投票（AI 不是队长）

- **镇民**：
    - 如果当前已经是第 3 任队长（这轮再否决深渊直接获胜），则强制投赞成。
    - 否则，如果 P_succ ≥ τ 则投赞成，反之反对。τ 初始 0.5，可根据之前连续失败次数降至 0.3。
- **深渊**：
    - 以概率 σ(β·(P_fail−0.5)) 投赞成，否则反对。可加入 10% 随机反转防止被识破。

### 6.3 任务执行

- **镇民**：永远放置真爆桶。
- **深渊**：每个深渊玩家独立决策，不进行座位号协调（避免"完美配合"的作弊感）。具体规则：
  - **紧急情况**（镇民已得 2 分，再赢一波就获胜）：通常必须放假爆桶。但 Rerir 例外——当队伍中有另一个深渊队友（队友会放假爆桶兜底），且深渊尚未到赛点（得分 < 2），且 Rerir 对 Columbina 的身份有足够信心（confidence ≥ 0.6）时，Rerir 会放真爆桶，让队友独自破坏任务来掩护自己身份，赌猎月翻盘。
  - **第四波特殊局**（7 人及以上对局，需要 2 枚假爆桶得分）：如果队伍中有深渊队友（凑够 2 枚假桶），则放假爆桶；如果是独狼，反正 1 枚假桶也无法得分，不如放真爆桶保护身份。
  - **常规情况**：综合考虑暴露风险、角色特性和比分局势，公式为：

    ```
    fake_prob = base × (1 - exposure) × urgency
    ```

    **base**（角色基础意愿）：
    - 猎月人（Rerir）：0.85（激进破坏者）
    - 赝月（Dottore）：0.50（需要维持"月神"伪装）
    - 普通深渊：0.70

    **exposure**（暴露风险）：
    - 基础值 = `1 / 队伍人数`（2 人队 = 0.5，5 人队 = 0.2）
    - 如果队伍中有深渊队友，暴露风险降至 30%（队友也会被怀疑，分散注意力）
    - 核心逻辑：队伍越小、越是独狼，放假桶越容易被锁定

    **urgency**（紧迫度）：
    - 镇民领先 1 分以上：1.2（落后了，加大破坏力度）
    - 深渊领先 1 分以上：0.8（领先了，可以保守隐藏身份）
    - 平局：1.0

    举例：Dottore 独自在 2 人队中，比分平局 → `0.50 × (1-0.5) × 1.0 = 0.25`，仅 25% 放假桶
    举例：Rerir 独自在 5 人队中，镇民领先 → `0.85 × (1-0.2) × 1.2 = 0.816`，约 82% 放假桶
    举例：Rerir 与队友同在 3 人队中 → `0.85 × (1-0.1) × 1.0 = 0.765`，但队友也会放，两个加起来几乎必破坏
