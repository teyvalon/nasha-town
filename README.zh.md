# 提瓦隆：守护祈月之夜

提瓦隆是一款《原神》千星奇域的社交推理游戏，其灵感来自于 Don Eskridge
的经典桌游[《抵抗组织：阿瓦隆》](https://boardgamegeek.com/boardgame/128882/the-resistance-avalon)。

* [美区及其他海外玩家见英文版](README.md)

## 上手指南

提瓦隆通常需要 5-8 名玩家。人数不足时，电脑玩家会自动加入游戏，保证每局至少有 5 名玩家。

## 基本规则

所有玩家分成两个阵营：

* 那夏镇阵营：守护祈月之夜，保护「月神」哥伦比娅
* 深渊阵营：破坏祈月之夜，猎杀「月神」哥伦比娅

每个人的身份都是保密的。深渊阵营的玩家互相知道彼此，但那夏镇阵营的玩家并不知道。特殊角色会得到额外信息，详见后面的「特殊角色」章节。

阵营人数根据对局总人数分配：

| 总人数 | 那夏镇 | 深渊 |
|-----|-----|----|
| 5 | 3 | 2 |
| 6 | 4 | 2 |
| 7 | 4 | 3 |
| 8 | 5 | 3 |

游戏中，将有 5 波狂猎袭击那夏镇，玩家可以使用元素视野技能（或长效视野技能）透过深渊迷雾看到它们。所有玩家依次**担任队长**进行组队，成行的队伍须冲入深渊迷雾并安置爆桶，以消灭来犯的狂猎、守护祈月之夜。每轮狂猎的数量取决于对局总人数：

| 总人数 | 第 1 波 | 第 2 波 | 第 3 波 | 第 4 波 | 第 5 波 |
|-----|-----|-----|-----|-----|-----|
| 5 | 2 | 3 | 2 | 3 | 3 |
| 6 | 2 | 3 | 4 | 3 | 4 |
| 7 | 2 | 3 | 3 | 4* | 4 |
| 8 | 3 | 4 | 4 | 5* | 5 |

面对每轮袭击，轮值队长应组建一支数量与狂猎相当的队伍。然后，所有玩家进行投票，选择赞成或反对这支队伍出击。只有当**超过半数**的玩家赞成时，组队才能获得通过。一旦成行，每位队员将发放一枚爆桶，去深渊迷雾中安置。若组队被否决，则由下一位轮值队长重新组队，并继续投票。每轮袭击都有最多 3 次组队机会（可以在游戏开始前的小屋内配置）。如果所有组队都被否决，深渊阵营将立即获胜，游戏结束。

每枚爆桶只能击杀一只狂猎。深渊阵营成员可能会用假爆桶掉包，而那夏镇阵营只能安置真爆桶。假爆桶无法击杀狂猎，因此狂猎能穿过深渊迷雾。此外，「队长」卡皮塔诺派出了两名守卫，帮助守护那夏镇。如果有狂猎穿过深渊迷雾，守卫就会消灭它们，但代价是一名守卫会被深渊感染并退场休息。

如果所有守卫都退场了，而仍有狂猎进入那夏镇，则深渊阵营获胜。否则，如果剩余的狂猎波次不足以突破防御，则那夏镇获胜（除非游戏中有「猎月人」，参见「特殊角色」章节）。这基本上等价于《抵抗组织》中的 5 回合任务，五局三胜定输赢。

`*` 在 7 人或以上的游戏中，「队长」会在第 4 波增派援军，因此在这一轮中，深渊阵营必须让两只或更多狂猎通过，才能得分。无论输赢，援军都会在第 4 波结束后退场休息。

## 特殊角色

目前有 3 种游戏模式可选：

1. **基础局**：无特殊角色，基本上等同于原版《抵抗组织》
2. **预言局**：那夏镇有「月神」哥伦比娅，深渊阵营有「猎月人」雷利尔
3. **迷雾局**：那夏镇有「月神」和「咏月使」菈乌玛，深渊阵营有「猎月人」和「赝月」多托雷

### 「月神」与「猎月人」

如果那夏镇率先赢得 3 分，且「猎月人」在深渊阵营中，那么游戏不会立即结束。「猎月人」将有一次机会，在所有镇民中找出真正的「月神」（即「猎月」）。如果「猎月人」找到了真正的「月神」，则深渊阵营反败为胜。

「月神」哥伦比娅在游戏开始时，知道谁是深渊玩家。但是因为有「猎月人」的威胁，她必须隐藏自己的身份。因此，「月神」玩家应当聪明地引导那夏镇玩家取胜，而不是大张旗鼓地引起「猎月人」的注意，从而最终输掉游戏。

### 「赝月」与「咏月使」

游戏开始时，「咏月使」菈乌玛会看到两位玩家都拥有三月权能，但她不知道谁是真正的「月神」哥伦比娅，谁是「赝月」多托雷。此时，「赝月」玩家会假扮「月神」来误导那夏镇玩家，以此取胜。所以，「咏月使」玩家必须凭借智慧，判断谁是真「月神」，并带领队伍守护那夏镇。

## 命令行模拟器

项目内置了一个本地命令行游戏模拟器，用于测试和开发。你将作为唯一的真人玩家，其余席位由 AI 填充。

**环境要求：** [uv](https://docs.astral.sh/uv/getting-started/installation/)

**运行：**

```bash
uv run main.py
```

启动后会提示选择玩家人数（5-8）、游戏模式（基础局 / 预言局 / 迷雾局）、AI 版本（v1 规则型 / v2 贝叶斯均场型 / v3 贝叶斯联合分布型）、随机种子、每轮最大组队次数及角色选择。游戏过程中，终端右侧会实时显示 AI 信念面板（可通过 `--no-panel` 关闭）。算法详见 [`ai-v1.md`](ai-v1.md)、[`ai-v2.md`](ai-v2.md) 和 [`ai-v3.md`](ai-v3.md)。

**平衡性测试：**

```bash
uv run benchmark.py             # 默认：v3, 500 局
uv run benchmark.py v1 100      # v1 AI, 100 局
uv run benchmark.py v3 1000 8   # v3 AI, 1000 局, 8 进程
```
